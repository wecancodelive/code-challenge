# Sum of Digits in a Number using While Loop

## Requirements
Write program to sum the digits in a number n.
Note: Use While Loop

## Algorithm in plain English
1. INIT N = 1234
2. INIT ANSWER = 0
3. WHILE N > 0
4. ANSWER = ANSWER + (N % 10)
5. N = N / 10
6. END WHILE
7. PRINT ANSWER
