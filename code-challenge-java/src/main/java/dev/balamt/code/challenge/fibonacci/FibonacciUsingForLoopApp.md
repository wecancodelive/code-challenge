# Print Fibonacci Using Simple For Loop


## Requirements
Write program to print fibonacci until n.
where n is the count of fibonacci numbers we wanted.
Note: Use plain for loop

## Algorithm in plain English
1. INIT N = 20
2. INIT START = 0
3. INIT FIRST = 1
4. INIT TEMP
5. PRINT START
6. PRINT FIRST
7. FOR I = 2 TO N
8. TEMP = START + FIRST
9. START = FIRST
10. FIRST = TEMP
11. PRINT TEMP
